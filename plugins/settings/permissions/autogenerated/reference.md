## Default Permission

Default permissions for the plugin

#### This default permission set includes the following:

- `allow-global-base`
- `allow-vault-base`
- `allow-copy-vault`
- `allow-move-vault`
- `allow-set-vault-base`
- `allow-is-empty-or-missing-dir`
- `allow-obsidian-vaults`
- `allow-path`
- `allow-load`
- `allow-save`
